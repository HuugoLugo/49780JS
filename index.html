Segunda Entrega

Calcular el costo total de productos seleccionados por el usuario

Se debe buscar incluir lo visto hasta la clase 8 (funciones).

clases, arrays

  • Se utilizan clases para "seccionar" los productos que se pretenden incluir, así como los códigos de descuento.
  • De modo que al tener una mayor cantidad de productos pueda existir un mayor, orden y se puedan ir agregando.
  • Los arrays se van a utilizar para poder buscar más facilmente el contenido de las clases.

variables

  • Se hace uso de variables let y const.
  • Solo se declaran algunas, ya que se le asigna valores después.
  • Se asignó un valor cero a una, de manera que de allí inicie la suma de valores.

if, switch, function

  • Se usa if en este caso para que al tener una respuesta de si, se ingrese código de descuento y se aplique el switch.
  • La función que se crea es para calcular el costo al aplicarse un código de descuento.
  • Se utilizan los "case" en este caso como productos y códigos de descuento.
  • Que en caso de seleccionarse, asignan cierto valor a variables que fueron declaradas previamente.

prompt, alert, console

  • Prompt para hacer que el usuario conteste e indique algo.
  • Alert para mostrar resultados, respuestas.
  • Console para visualizar mejor lo que se hace.

do while

  • Se eligió para que ejecute, y al final a partir de la respuesta se repita o no.

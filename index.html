Actividades de Tarea Clase2

1-Determinar si un usuario puede acceder a una página web en base a dos condiciones: el usuario debe estar registrado y debe ser mayor de edad. Si se cumplen se muestra un mensaje de bienvenida, si no se cumple niega el acceso.

2-Determina si un producto está en stock, para estar en stock debe estar marcado como "en stock" o tener un precio con descuento. Si se cumple alguna de estas condiciones se muestra un mensaje de "producto disponible", de lo contrario muestra "producto agotado".

3-Verifica si un usuario ha iniciado sesión. Si no ha iniciado, que le muestre un recordatorio, si ya inició sesión que muestre un mensaje de bienvenida.

4-Simula un sistema de autenticación para un sitio web. Se debe solicitar ingresar un nombre de usuario y una contraseña. El nombre de usuario debe ser "admin" y la contraseña debe ser "secreta" o "123456".

Conceptos Generales

Arrays

Tipo de dato para almacenar valores en forma de lista.

Característica y uso

  • Puede contener números, strings, booleanos, objetos, incluso una lista de listas.
  • Es posible agregar o quitar elementos en todo momento.
  • Tienen un índice que comienza siempre desde la posición cero.
  • El orden en que se coloquen si importa.
  • Para declarar un array hay que comenzar con una variable, se abren corchetes [], dentro lo que coloquemos debe estar separado por una coma. Se puede declarar un array vacío y más adelante incorporar elementos.
  • Podemos acceder a un elemento en específico del array a través de su posición. Se coloca el nombre de la variable que contiene el array, se abren corchetes y dentro se coloca el número de la posición del elemento / valor que deseamos.
  • Para recorrer un array hacemos uso del bucle for.

Arrays (Propiedades - Métodos)

Las propiedades y métodos se utilizan en el array para conocer cierta información, añadir o quitar elementos, generar nuevos arrays o incluso mezclarlos.

variable(que contiene al array).propiedad / método (al final del método se colocan parentesis())

  • Propiedad length, utilizada para conocer el largo de un array, identificar cuantos elementos tiene, en cadenas de texto nos dice cuantos caracteres contiene.
  • Método push, permite agregar un nuevo elemento al final del array, se coloca dentro de los parentesis el elemento a agregar.Si se desea agregar al inicio del array se utiliza unshift.
  • Método pop, elimina el último elemento del array, no se coloca nada en los (). Mientras que shift elimina el primer elemento del array.
  • Método splice, permite eliminar uno o más elementos de un array, funciona con dos parámetros: índice(en donde se posiciona) y cantidad de elementos a eliminar(desde esa posición).
  • Método join, genera un string con los elementos del array separados por el valor que colocamos de parámetro (por ejemplo, una coma, un espacio, un asterisco).
  • Método split, permite convertir una cadena de caracteres a un array, se coloca en los () lo que separa a los elementos, por ejemplo un espacio o una coma.
  • Método concat, permite combinar dos arrays en uno, en el parámetro se coloca el array que se va a combinar.
  • Método indexOf, permite obtener el índice (posición) de un elemento, que colocamos como parámetro. Si elemento no existe retorna -1.
  • Método includes, permite saber si un elemento (que se coloca como parámetro) existe o no dentro de un array, retorna true / false.
  • Método reverse, invierte el orden de los elementos del array, no se coloca nada en los ().

Arrays de objetos

Estructura de datos para coleccionar objetos.

  • Podemos asignar objetos literales o previamente instanciados en la declaración del array.
  • Podemos agregar nuevos objetos usando el constructor, dentro de los corchetes del array se abren llaves, dentro está el objeto (recordando que debe ser clave: valor), se cierran llaves, para añadir más objetos solo basta separarlos con una coma.
  • Para agregar nuevos objetos con el método push, dentro de los corchetes se abren llaves que contienen las clave: valor, separados por una coma.

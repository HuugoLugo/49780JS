Operadores

Los operadores son utilizados para comprobar valores y evaluar condiciones.

Operadores relacionales

  • == Igualdad, se fijará si los valores son iguales.
  • === Igualdad estricta, se fijará si los valores son iguales, y además que sean el mismo tipo de dato.
  • != Distinto que, se fijará en que los valores sean diferentes.
  • !== Distinto estricto, se fijará en que los valores sean diferentes, además de distinto tipo de dato.
  • > Mayor que, >= Mayor o igual que. Menor que, Menor o igual que.

Operadores lógicos

  • && Operador AND, y. Este va a devolver verdadero, si las dos condiciones son verdaderas, devuelve falso si al menos una condición es falsa.
  • || Operador OR, o. Este va a devolver verdadero, si al menos una de las condiciones es verdadera, devuelve falso si las dos condiciones son falsas.
  • ! Operador NOT, no. Este invierte el valor de ala condición, por ejemplo si tenemos verdadero, este operador lo convertirá en falso.

Condicionales

Nos permiten realizar algo, en caso de que se cumpla algo. Tomar decisiones a partir de condiciones.

if, si se cumple una condición realiza una operación.

else, si no se cumple el if, con else le pedimos que haga otra operación.

else if, podemos con esto añadir más condiciones en caso de que no se cumpla el if inicial. Anidar condiciones.

Podemos usar else if cuantas veces lo necesitemos, pero solo puede haber un if al inicio y un else al final.

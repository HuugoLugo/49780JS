Conceptos Generales

Variables

Las variables permiten asignar un valor, guardar información. Pueden ser: "let" permite cambiar valor, o "const" permite que el valor se mantenga.

Declarar una variable significa crearla.

  • let o const, seguido del nombre que queremos asignarle, usaremos "camelCase" en variables let, y mayúsculas en const.
  • Todas las variables deben iniciar con una letra, y son keysensitive.
  • No debemos usar espacios (se usa guión bajo), ni caracteres especiales.

Asignación de una variable significa darle un valor después del signo de igual.

  • Los valores pueden ser: númericos, cadena de texto, booleanos, null, undefined.
  • Los números no van entre comillas, las cadenas de texto si

La inicialización de una variable significa realizar la declaración y la asignación al mismo tiempo.

Prompt, consola y alert

Prompt muestra un cuadro de dialogo para que el usuario ingrese un dato.

Alert muestra una ventana sobre el sitio web

Console (console.log) muestra el mensaje o acción a la consola del navegador web.
